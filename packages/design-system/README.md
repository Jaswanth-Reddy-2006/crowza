# Design System

A shared library of UI components and hooks for the Venue Experience Platform.

## Components

- `Button`: Primary, secondary, and tertiary buttons.
- `Card`: Container for grouping information.
- `TextField`: Input fields for forms.
- `Alert`: Contextual feedback for users.
- `Dialog`: Modals for critical information or actions.
- `Badge`: Indicators for status or counts (e.g., wait times).

## Styles

- `global.css`: Base styles and CSS variables.
- `typography.css`: TYPOGRAPHY tokens and scales.
- `accessibility.css`: Focus states and accessibility overrides.

## Usage

Components support both React (Web) and React Native (Mobile) where appropriate, using a universal approach.
